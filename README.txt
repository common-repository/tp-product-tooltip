== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate TP Woocommerce Product Tooltip through the 'Plugins' menu in WordPress.

TP Woocommerce Product Tooltip will add a new menu item called ‚ÄúTP Woocommerce Product Tooltip‚Äù in wordpress menu. Here you are able to configure all the plugin settings.
